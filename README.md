An intelligent Telegram bot for managing Google Calendar events using AI-powered natural language processing.

## ✨ Features

- ✅ **Create Events**: Add calendar events using natural language (text or voice)
- 🗑️ **Delete Events**: Remove events with simple commands
- ✏️ **Update Events**: Modify existing calendar events
- 📋 **View Events**: List upcoming events for any date range
- 🎤 **Voice Support**: Send voice messages in any language
- 🧠 **AI-Powered**: Understands context and natural language in Persian and English
- 💬 **Conversational Memory**: Remembers conversation context
- 🌍 **Multi-language**: Works with both Persian and English inputs

## 🎯 Use Cases

- Personal calendar management via Telegram
- Voice-based scheduling for hands-free operation
- Quick event creation while on the go
- Team calendar coordination
- Meeting reminder system

## 📋 Prerequisites

- [n8n](https://n8n.io) (self-hosted or cloud version)
- Google account with Calendar API access
- Telegram Bot Token from [@BotFather](https://t.me/botfather)
- OpenAI API Key
